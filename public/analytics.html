Choose and implement an analytics solution to track visitor engagement on Forward Horizon.

Option 1: Google Analytics 4 (GA4) - Recommended

Setup Instructions

Step 1: Create Google Analytics Account
  1. Go to Google Analytics
  2. Click "Start measuring"
  3. Account name: "Forward Horizon"
  4. Property name: "theforwardhorizon.com"
  5. Select your timezone and currency
  6. Choose business size and objectives
Step 2: Get Your Measurement ID

After setup, you'll receive a Measurement ID like: G-XXXXXXXXXX

Step 3: Add to Your Website

Add this code to the <head> section of all HTML files:

<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
window.dataLayer = window.dataLayer || [];
function gtag(){dataLayer.push(arguments);}
gtag('js', new Date());
gtag('config', 'G-XXXXXXXXXX');
</script>

Plausible Analytics

Setup:
  1. Sign up at Plausible.io ($9/month)
  2. Add your domain
  3. Add this single line to your HTML:
<script defer data-domain="theforwardhorizon.com" src="https://plausible.io/js/script.js"></script>

What to Track

Key Metrics for Forward Horizon:
  • Page Views: Which pages get the most traffic
  • Form Submissions: Application and contact form completions
  • User Sources: Where visitors come from (Google, social, direct)
  • Device Types: Mobile vs desktop usage
  • Geographic Data: Where your visitors are located
  • Bounce Rate: How many leave without interaction
  • Session Duration: How long people stay on site

Quick Implementation Script

Add this to all your HTML files (replace G-XXXXXXXXXX with your ID):
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
window.dataLayer = window.dataLayer || [];
function gtag(){dataLayer.push(arguments);}
gtag('js', new Date());
gtag('config', 'G-XXXXXXXXXX', {
anonymize_ip: true,
cookie_flags: 'SameSite=None;Secure'
});

// Track form submissions
function trackFormSubmit(formType) {
gtag('event', 'form_submit', {
'form_type': formType,
'page_location': window.location.href
});
}
</script>
